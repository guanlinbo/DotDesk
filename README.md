# \# DotDesk

# 

# DotDesk 是一个基于 C# / WebRTC 的高性能远程桌面项目，目前仍在持续开发中。

# 

# 这个项目最开始只是为了自己学习远程控制、音视频传输、WebRTC 和桌面采集相关技术，后来慢慢开始往真正可用的远程桌面方向演进。

# 

# 目前已经实现：

# 

# \- WebRTC P2P 连接

# \- STUN / TURN NAT 穿透

# \- H264 视频传输

# \- DXGI / GDI 屏幕采集

# \- DataChannel 输入控制

# \- 信令服务器

# \- 临时密码验证

# 

# 后续计划：

# 

# \- DXGI DirtyRect 脏矩形优化

# \- GPU 硬编码（NVENC / AMF / QSV）

# \- 多显示器支持

# \- 文件传输

# \- 剪贴板同步

# \- 音频传输

# \- 自适应码率

# \- 更完整的 UI

#
